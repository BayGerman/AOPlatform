# AOPlatform

AOPlatform is the developer landing page for the Agency Online architecture.

## Platform Structure

AOPlatform
│
├── AOBrainV1_2
│   Business knowledge authoring
│   Workbook, dictionaries, templates, skills
│
├── AOBrainV2
│   Runtime business graph
│   Standard Library, Business Discovery, Working Context
│
├── AOBrainV3
│   Capability Engine
│   Capabilities, Business Blocks, ArcLago package
│
└── ArcLago
    Customer UI and runtime platform
